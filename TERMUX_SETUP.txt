# TERMUX SETUP GUIDE - Mesh Chat

## Step 1: Install Termux

1. Open **Google Play Store** on your Android phone
2. Search for **"Termux"**
3. Install **Termux** (by Fredrik Fornwall)
4. Open Termux app

---

## Step 2: Setup Termux (Run Once)

Copy and paste these commands in Termux:

```bash
pkg update -y && pkg upgrade -y
pkg install python git -y
```

Wait for installation to complete (2-5 minutes).

---

## Step 3: Download Mesh Chat

```bash
cd ~
git clone https://github.com/Muneeb50702/B2B_Communication.git
cd B2B_Communication
```

---

## Step 4: Connect Phones to Same WiFi

**IMPORTANT:** All phones must be on the **same WiFi network**!

- Connect all phones to your home WiFi
- Or create hotspot on one phone, others connect to it

---

## Step 5: Run on All Phones

On **EACH phone** in Termux:

```bash
cd ~/B2B_Communication
python mesh_chat_termux.py
```

---

## Step 6: Use the Chat

### Wait 3-5 seconds, then:

**1. See who's online:**
```
>>> /list
```

**2. Select recipient:**
```
>>> /send User_ABC123
```

**3. Send message:**
```
>>> Hello from my phone!
```

**4. Exit:**
```
>>> /quit
```

---

## ONE-LINE SETUP (After First Time)

Next time, just run:

```bash
cd ~/B2B_Communication && python mesh_chat_termux.py
```

---

## Troubleshooting

### "No users found"
- Make sure all phones are on same WiFi
- Wait 10 seconds
- Try `/list` again

### "Permission denied"
- Run: `termux-setup-storage`
- Allow storage access

### "Module not found"
- Run: `pkg install python -y`

---

## Quick Reference

```
/list              - Show online users
/send User_ABC123  - Select recipient
Type message       - Send to selected user
/quit              - Exit app
```
